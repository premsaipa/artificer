# Artificer

Building IoT and embedded systems from foundation to autonomy.

## About

I'm Premsai, a 23-year-old engineer based in India, working in IT support by day and learning to build connected devices by night. This repository documents my journey from Arduino hobbyist to IoT engineer, one project at a time.

I'm working toward two flagship builds:
- An autonomous house-cleaning robot
- A high-resolution audio player

Each project here is a rung on the ladder to those goals.

## Stack

- **Embedded:** ESP32, Arduino framework, PlatformIO
- **Protocols:** MQTT, I²C, SPI, I²S
- **Cloud:** AWS IoT, Adafruit IO
- **Tools:** VSCodium, Git, Ubuntu Linux

## Why this exists

I'm building in public for two reasons: to keep myself accountable, and to show real work to future employers and collaborators. Every project is documented with what I learned, what broke, and how I fixed it.
